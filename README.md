# Homebrew Tap for AiUsage

Install the macOS menu bar app and its Claude statusLine helper:

```bash
brew install --cask j3s30p/tap/aiusage
```

AiUsage `0.1.0` is an unsigned, unnotarized preview. After the first blocked launch, macOS may
require `System Settings → Privacy & Security → Open Anyway`. The cask does not bypass
Gatekeeper.

Project and usage documentation: <https://github.com/j3s30p/AI_Usage>
